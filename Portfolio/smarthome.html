Objective:

TThe objective of the Smart Home Control System project is to create an intuitive and user-friendly application using the Tkinter library in Python. This application allows users to remotely control and manage the lighting systems in different rooms of a smart home, leveraging the Philips Hue smart lighting system and the Phue Python library for seamless integration.

Results

Highlights of the project:

  • Individual Room Control:
     Enable users to control the lighting in specific rooms such as Salman's Bedroom, Ahmed's Bedroom, and the Living Room independently.
     Functions include turning lights on/off, adjusting brightness, and implementing dynamic dimmer settings.

  • Provide a convenient option for users to control all lights collectively with a single click, offering a streamlined experience when leaving or entering the home.

  • Incorporate a real-time status overview section that displays the current state of each room's lighting, including whether the lights are on or off, and the brightness level.

  • Introduce an engaging party mode that dynamically changes the colors of the lights for a user set amount of time

  • Implement a color-changing feature allowing users to personalize the lighting ambiance in each room.

  • Incorporate the Tkinter library to facilitate the creation of an interactive and user-friendly graphical user interface.
